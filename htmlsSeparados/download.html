Sobre Minha Conta

Como recomendação (ou não, porque recomendamos windows), utilizaremos o Ubuntu, uma das distribuições de Linux mais famosas. Há uma razão para todo esse sucesso: o sistema é de simples utilização, e até mesmo iniciantes no mundo do software livre gostam dele. Além disso, é uma boa opção para quem tem uma máquina fraca e problemas de desempenho por ser mais leve que a concorrência.

Antes de iniciar a instalação do Ubuntu, recomendamos, no mínimo:

  1. 25 GB de espaço livre para instalar o sistema;
  2. 4GB RAM;
  3. Processador dual core pelo menos 2GHz;
  4. Porta USB livre;
  5. Pen drive com capacidade de pelo menos 8 GB (se tiver 12 ou mais é melhor);
