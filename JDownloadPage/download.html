Sobre Minha Conta

Como recomendação (ou não, porque sempre recomendamos windows), utilizaremos o Ubuntu, uma das distribuições de Linux mais famosas. Há uma razão para todo esse sucesso: o sistema, apesar de conter todas as ferramentas necessárias, é de simples utilização, e até mesmo iniciantes no mundo do software livre gostam dele. Além disso, é uma boa opção para quem tem uma máquina fraca e problemas de desempenho por ser mais leve que a concorrência.

Antes de iniciar a instalação do Ubuntu, recomendamos, no mínimo:

  1. 25 GB de espaço livre para instalar o sistema;
  2. 4GB RAM;
  3. Processador dual core pelo menos 2GHz;
  4. Porta USB livre;
  5. Pen drive com capacidade de pelo menos 8 GB (se tiver 12 ou mais é melhor);

O tempo estimado para a instalação é de 1 hora. A seguir, iniciaremos a configuração do Ubuntu e a instalação do sistema operacional:

O Ubuntu perguntará se você deseja testá-lo primeiro, ou se deseja instalá-lo em seu computador: Escolha a opção "Instalar o Ubuntu, nessa fase também se define a linguagem do seu sistema:

3. Determine o idioma de seu teclado

Nessa etapa, você escolherá a linguaguem do teclado

4. Configure sua instalção

É preciso optar por uma instalação completa, com todos os programas do sistema, ou a mínima, com apenas utilitários básicos e navegador, tendo também a opção de instalar o que você deseja posteriormente.

5. Escolha o disco ou compartição

Finalmente, é hora de escolher em que disco do seu computador será instalado o Ubuntu. A opção padrão é apagar tudo e instalar o sistema. ATENÇÃO: se você possui Windows, macOS ou outrosistema Linux instalado, ele será deletado caso você opte pelo caminho default. A alternativa é criar ou modificar as partições. Esse é um processo, o qual merece muito cuiadde, pois continua com o risco de perder dados anteriormente salvos. Nesse tutorial, optamos pela primeira opção, por ser a mais simples, mas a segunda opção é a mais recomendada para quem quer manter dois sistemas operacionais instalados no mesmo dispositivo, como dual boot de Windows e Linux.

6. Criptografe (ou não) seu disco

Neste próximo passo, você poderá optar novamente por criptografar os dados salvos no disco. Se escolher esse caminho, crie uma chave de segurança e uma chave de recuperação, elas serão indispensáveis.

Configure o relógio de acordo com o padrão de sua região
